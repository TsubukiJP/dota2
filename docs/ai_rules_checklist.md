# 最終確認（出力前チェック）
**重要**: 翻訳結果のみをJSON形式で出力する前に、以下を確認すること。

## 技術的チェック（必須）
- [ ] 変数・タグが原文と完全一致（大文字小文字、先頭 `%` 含む）しているか？
- [ ] JSONキーが原文と完全一致しているか？
- [ ] 改行 `\n` の位置と個数が維持されているか？

## 翻訳チェック
- [ ] `glossary.csv` の用語ルールを適用しているか？
- [ ] KEEP指定以外の英単語が混入していないか？
- [ ] プロファイル固有のスタイル（常体/丁寧語、体言止め等）を適用しているか？
- [ ] `*_Lore`がある場合、直訳せず、世界観を尊重し、重厚かつ自然な文体で翻訳しているか？
