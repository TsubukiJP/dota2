# イベント翻訳ルール
Tooltip系以外のイベント/特殊モードテキスト専用ルール。

## A. Eventデフォルトルール

### 1. イベントの雰囲気を尊重
*   イベント固有のメッセージ（HUD表示、アナウンス、ゲームルール説明など）は、**そのイベントの世界観**に合わせた自然な日本語にする。
*   例: Diretide → 少しユーモラス/ホラー風も許容
*   例: Aghanim's Labyrinth → 冒険・RPG的な雰囲気

### 2. HUDメッセージ・トースト
*   `*_Toast`, `DOTA_HUD_*` などの短いメッセージは、臨場感を重視。
*   過度に硬い表現は避け、プレイヤーが瞬時に理解できる簡潔さを優先。

### 3. ゲームルール説明
*   `gameplay_tip_*` などのチュートリアル/Tip系は、**丁寧語（です・ます調）を許容**。
*   プレイヤーへの説明として自然な文体を選択。

### 4. ユニット名・アイテム名
*   イベント固有のユニット名（`npc_dota_greevil` など）やアイテム名は、**原則英語維持**。
*   ただし、`glossary.csv` に登録されている場合はそちらに従う。
